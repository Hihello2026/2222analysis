# 📊 منصة التحليل الكمي للأسهم السعودية (Quantitative Analysis Platform)

مشروع استثماري يعتمد على **نموذج ماركويتز (Modern Portfolio Theory)** لتحسين توزيع الأصول بين سهمي **مسار (جبل عمر)** و **المطاحن العربية**.

## 🚀 المميزات
* **بيانات حية:** سحب بيانات السوق السعودي لحظياً عبر `yfinance`.
* **تحسين المحفظة:** حساب التوزيع الأمثل للأصول لتقليل المخاطر (Minimizing Volatility).
* **واجهة تفاعلية:** لوحة تحكم متكاملة مبنية باستخدام `Streamlit`.

## 🛠️ التقنيات المستخدمة
* **Python:** لغة البرمجة الأساسية.
* **PyPortfolioOpt:** للمحاكاة الرياضية والنماذج الكمية.
* **Pandas:** لمعالجة السلاسل الزمنية المالية.

## 📈 الهدف من المشروع
تطبيق المنهجيات المتبعة في فرق **Systematic Equity** لتحقيق أقصى قدر من الكفاءة الإحصائية في إدارة المحافظ.
